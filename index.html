Data Journalism Center

Welcome to Data Journalism Center!

"데이터 저널리즘(data journalism)"은 저널리즘(journalism)과 컴퓨터 공학(computer science)을 결합한 학제 간 뉴스생산 과정으로 'data-driven Journalism', 'computational journalism', 'open source journalism', 'journalism as programming', 'programming as journalism' 같은 흥미로운 용어들로 표현합니다.

이 저널리즘을 담당하는 기자들을 뜻하는 용어도 'data journlaist', 'hacker journalist', 'computational journalist', 'database editor', 'jourveloper', 'progojournalist' 로 다양합니다.

미국과 유럽, 홍콩의 대학들은 이미 저널리즘 수업과 컴퓨터 공학 수업을 학부와 대학원에 함께 제공합니다. 서강대학교 지식융합미디어학부에서 다음 과목들을 개설했습니다.

학부: 데이터 뉴스 , 텍스트 마이닝

일반대학원 : 데이터 저널리즘, 텍스트 마이닝 기법

언론대학원: 빅데이터와 디지털 뉴스, 텍스트 마이닝과 시각화

이 과목들은 임종섭 교수가 담당합니다.

--

데이터 저널리즘 센터는 데이터 저널리즘의 의미, 언론의 역할, 대안언론으로서의 가능성 등을 실무차원과 이론차원에서 고민하는 공간입니다. 데이터 저널리즘 현상을 이해하려면 '임종섭 (2015). 데이터 저널리즘. 서울: 커뮤니케이션북스'를 참고하세요.
